# Escape from Max - Design Document

## 1. 做给谁（目标用户）
喜欢恐怖生存类小游戏的休闲玩家。想在浏览器中体验被怪物追杀的紧张刺激感，无需下载安装，打开即玩。适合快速解压、朋友间互相挑战生存时长。

## 2. 解决什么问题
- 市面上缺少轻量、免费、浏览器即开的恐怖追逐游戏
- 大多数恐怖游戏需要下载、配置要求高
- 需要一个快速上手、紧张刺激的休闲小游戏
- 在别墅环境中加入追逐玩法，既有探索感又有压迫感

## 3. 核心功能列表

### 3.1 别墅地图
- 俯视视角 2D 别墅，包含多个房间：
  - 客厅（大房间，有沙发、茶几等家具）
  - 餐厅（有餐桌、椅子）
  - 厨房（有灶台、冰箱）
  - 走廊（连接各个房间）
  - 主卧（有床、衣柜）
  - 浴室
  - 书房（有书桌、书架）
- 房间之间通过门洞连接
- 家具作为障碍物，玩家和怪物都需要绕行

### 3.2 玩家操控
- WASD 移动，支持斜向移动
- 玩家比 Max 稍快，可以通过走位甩开
- 碰撞检测：不能穿过墙壁和家具

### 3.3 怪物 Max
- 红色眼睛在黑暗中发光
- 使用 BFS 寻路算法追踪玩家，会绕过墙壁和家具
- 速度略慢于玩家，但会逐渐加速
- 越靠近玩家，屏幕震动越强、心跳效果越明显
- 碰到玩家即死亡，游戏结束

### 3.4 手电筒/黑暗视野
- 整个别墅笼罩在黑暗中
- 只有玩家周围有手电筒光照范围
- Max 的眼睛在光照范围内可见
- 增强恐怖氛围和紧张感

### 3.5 游戏流程
- 开始画面：标题 + "Press SPACE to Start"
- 游戏开始：玩家随机出生在一个房间，Max 在远处
- 生存计时：显示存活时间
- 死亡画面：显示存活时长，可按 SPACE 重新开始

### 3.6 氛围效果
- 暗色调配色
- 手电筒光束中有浮尘粒子
- Max 靠近时画面边缘变红
- 屏幕震动（Max 很近时）
- 心跳脉冲视觉效果
